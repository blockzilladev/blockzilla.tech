File Structure

- css ( Template CSS files )
 
- fonts ( Template icon fonts )
 
P.S.
              - These fonts are for icons only.
              - The template use Google fonts.
              - If you want to change the font, pick any font you want from Google fonts and replace  in css/style.css file.                                    
                
     
 
- img ( Template images )
 
- js ( Template javaScript files )
 
- main.js ( Main javaScript file ).
 
- responsive.js ( Media Query  ).
 
- php ( contact.php - file for contact form).
 
P.S
 
Note: Contact form will not work offline. You must upload your website on a server with support of PHP and PHP mail() function.

HTML Structure

The template is based on Bootstrap v3.3.6 Framework - http://getbootstrap.com/
 
The default Bootstrap grid system utilizes 12 columns, making for a 1170px wide depending on your viewport.
 
Every line is commented to help you understand what that line do.
 
<div class="container">

<div class="row">

<div class="col-lg-12 col-md-12 col-sm-12 col-xs-12">

The content goes here ...

</div> <!-- End col-lg-12 -->

</div> <!-- End row -->

</div> <!-- End container -->

 

CSS Structure

bootstrap.min.css ( Generic styles, resets & normalization ).
 
style.css ( The basic structure styles ).
 
responsive.css (the media query )

Animation

Link to the CSS animation library

<link rel="stylesheet" href="css/animate.css">

 

Link and activate WOW.js

<script src="js/wow.min.js">
              <script>

              new WOW().init();

              </script>

 

 

<div class="wow slideInLeft" data-wow-duration="2s" 

    data-wow-delay="5s">

 </div>

 

data-wow-duration: Change the animation duration
data-wow-delay: Delay before the animation starts
data-wow-offset: Distance to start the animation (related to the browser bottom)
data-wow-iteration: Number of times the animation is repeated

Parallax Effect



	Put Class  parallax-image in body tag


<body class="parallax-image">



	and in style.css


/*  Parallax Image */

.parallax-image #home-section {

    padding: 0;
    background-repeat: repeat;
    background-attachment: fixed;
    background-position: center center;
    background-image: url(../images/background/hero-main.png);
    -webkit-background-size: cover;
    -moz-background-size: cover;
    -o-background-size: cover;
    background-size: cover;
}



	 



	and in main.js file


/*Parallax*/

        jQuery('.parallax-image #home-section').parallax('50%', 0.1);
